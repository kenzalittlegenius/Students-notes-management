# Gestion des notes des étudiants
# Students notes management

## Description du projet
## Project Description 
The goal of this project is to create classes representing students ('Student' class), notes ('Grade' class), results of a teaching unit (`TeachingUnitResult` class) and students promotions ('cohort' class).
Le but de ce projet est de créer des classes permettant de représenter des étudiants (classe `Student`), des notes (classe `Grade`), des résultats à une unité d'enseignement (classe `TeachingUnitResult`) et des promotions d'étudiants (classe `Cohort`).
